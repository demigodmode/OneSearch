# Document Preview

View full document content from search results.

    Detailed preview guide coming soon.

## Features

- **Syntax highlighting** - Code files with Prism.js
- **Markdown rendering** - Markdown documents render as formatted content
- **Readable text pages** - Long extracted text is split into manageable preview pages
- **Search match navigation** - Search terms are highlighted when you open a result from search
- **Image previews** - Indexed JPG, PNG, WebP, GIF, TIFF, and RAW photos can show authenticated previews
- **Photo metadata** - Camera, lens, ISO, aperture, exposure, focal length, date taken, and dimensions show when available
- **Comic details** - CBZ files show ComicInfo metadata and page lists
- **Media details** - Audio/video files show ffprobe metadata when available
- **Keyboard shortcuts** - Press Escape to go back
- **Metadata** - View file info and extracted metadata

Click any search result card to open the preview page.

!!! note "Preview limits"
    Admins can disable previews or set preview size limits in Admin → Settings → File Previews. RAW previews use embedded JPEGs when available; OneSearch does not decode RAW sensor data.
