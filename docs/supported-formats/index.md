# Supported File Formats

OneSearch supports multiple file formats with specialized extractors.

## Currently Supported

| Format | Extensions | Features |
|--------|------------|----------|
| **Text Files** | .txt, .log, .conf, .cfg, .ini | Encoding detection, full-text search |
| **Markdown** | .md, .markdown | YAML front-matter, full-text search |
| **PDF** | .pdf | Text extraction, metadata |
| **Word** | .docx | Paragraphs, tables, metadata |
| **Excel** | .xlsx | All sheets, cell values |
| **PowerPoint** | .pptx | Slides, speaker notes |
