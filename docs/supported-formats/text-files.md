# Text Files

Plain text file support in OneSearch.

!!! note "Coming Soon"
    Detailed format documentation coming soon.

## Supported Extensions

- .txt
- .log
- .conf, .cfg, .ini
- Other plain text files

## Features

- Automatic encoding detection (UTF-8, Latin-1, etc.)
- Full-text indexing
- Size limits configurable via `MAX_TEXT_FILE_SIZE_MB`
