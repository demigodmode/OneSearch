## Required Variables

- `MEILI_MASTER_KEY` - Meilisearch API key

## Optional Variables

- `DATABASE_URL` - SQLite database path
- `LOG_LEVEL` - Logging level (DEBUG, INFO, WARNING, ERROR)
- `MAX_TEXT_FILE_SIZE_MB` - Max text file size (default: 10)
- `MAX_PDF_FILE_SIZE_MB` - Max PDF file size (default: 50)
- `MAX_OFFICE_FILE_SIZE_MB` - Max Office file size (default: 50)

See `.env.example` in the repository for all options.
