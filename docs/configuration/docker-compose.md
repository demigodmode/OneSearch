## Volume Mounts

OneSearch can only index directories that are mounted into the container. There are a few ways to approach this: mounting specific paths, parent directories, or your whole filesystem. Each has different tradeoffs.

See [Volume Mounts](volume-mounts.md) for the full breakdown and examples.
