# Frontend Development

Develop the OneSearch frontend.

!!! note "Coming Soon"
    Comprehensive frontend development guide coming soon.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Project Structure

```
frontend/
├── src/
│   ├── pages/       # Page components
│   ├── components/  # Reusable UI components
│   ├── lib/         # API client, utilities
│   └── types/       # TypeScript types
└── package.json
```

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- TanStack Query for data fetching
- shadcn/ui for UI components
- Tailwind CSS for styling

## Adding Features

1. Create components
2. Add API client functions
3. Update types
4. Style with Tailwind
5. Test in browser

See the README for detailed guidelines.
